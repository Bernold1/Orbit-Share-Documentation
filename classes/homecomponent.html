receiveSearchContent

  • receiveSearchContent(): void
    • Defined in src/app/components/home/home.component.ts:68

    recieveSearchContent() is a function in the home.component. This function is responsible for loading the search results. This function is used along with the toolbar.component and the search.service. Here we subscribe to the results emitted as an observable from the toolbar.component using the searc.service. After recieving an update on the subscribed value, the results are loaded into the newestContentArr which results in the view being updated. This happens by overwriting the value newestContentArr which is used in the home.component template.

    Returns void
