addTag

  • addTag($event: Event): void
    • Defined in src/app/modules/content/upload-content/upload-content.component.ts:106

    Gets the input from the tag input-field, extracts the value, removes the content in the field so its ready for a new tag. Takes the value of the input and sets the value into a new array

    ... in front of this.tagsArray.value is a spread operator which spreads the items inside of this.tagsArray.value which makes a copy of this.tagsArray.value where the value is than pushed to the "tagName" field. Using setValue causes the required validator to recognize that the field has indeed been used and has an input, since a new array is being created and the value of the input set and pushed into the new array.

    Parameters

    • $event: Event

    Returns void
