onSubmit

  • onSubmit(content: TempContent): void
    • Defined in src/app/modules/collection/upload-collection/upload-collection.component.ts:80

    ** function bound to child (ContentComponent) event.** Upon submitting the collection form a new ContentText object is created based on the input emitted from the content component. Here the content text and file is added to each of their seperate arrays. The service is than called to persist the file, here we are subscribed to the observable, where upon a sucessful http request the subscribe method takes the response data of type FileResponse, where each filename in the response is mapped to the collection text array. Here a spread operator is used to add the file names to each object inside of the collectionText array. After having adding the file names to each object, the service is once again called passing the text from each content (collectionText array), and routing the user to the home page.

    Parameters

    • content: TempContent

    Returns void
